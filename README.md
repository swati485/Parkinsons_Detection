# 🧠 Parkinson’s Disease Detection Using Machine Learning  

## 📌 Project Overview  
This project was developed as part of a hackathon and focuses on detecting Parkinson’s disease using machine learning and computer vision techniques applied to spiral and wave drawings. The goal is to analyze patterns in these drawings, extract meaningful features, and classify whether a person may have Parkinson’s disease.  

The workflow includes:  
- Preprocessing spiral and wave images  
- Feature extraction using computer vision techniques  
- Classification with machine learning models  
- Model evaluation and visualization of results
